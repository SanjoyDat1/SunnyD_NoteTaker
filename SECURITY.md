# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

Please open a GitHub Issue to report security concerns. Do not commit API keys or secrets to the repository.

**Note:** Your OpenAI API key is stored only in your browser's `sessionStorage` and is never sent to any server except OpenAI's API.
