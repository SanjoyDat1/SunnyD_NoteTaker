# Contributing to SunnyD Notes

Thanks for your interest in contributing!

## Development Setup

```bash
npm install
npm run dev
```

## Code Style

- Use consistent formatting (the project uses standard JSX/JS conventions)
- Keep components focused; `sunnyd.jsx` contains the main app logic

## Pull Request Process

1. Ensure the app runs (`npm run dev`) and builds (`npm run build`)
2. Describe your changes clearly in the PR
3. Reference any related issues

## Areas for Contribution

- Bug fixes
- UI/UX improvements
- New suggestion types
- Performance optimizations
- Documentation
